# Assets du jeu

## Instructions pour ajouter l'image de la map

1. Sauvegardez l'image de la map que vous avez partagée
2. Nommez-la `arlong_park_map.png` ou `arlong_park_map.jpg`
3. Placez-la dans ce dossier `assets/`

Le jeu est configuré pour charger automatiquement cette image au démarrage.

## Structure actuelle
```
assets/
  - arlong_park_map.png  (à ajouter)
  - README.md (ce fichier)
```
